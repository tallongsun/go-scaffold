# go-scaffold

## build
```
git clone https://github.com/tallongsun/go-scaffold.git
cd go-scaffold
export GO111MODULE=on
go build cmd/demo/main.go 
RUN_MODE=alpha ./main
```

## module management
```
git clone https://github.com/tallongsun/go-scaffold.git
cd go-scaffold
go mod init github.com/tallongsun/go-scaffold
```
